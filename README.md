# CatPhotoApp

A simple HTML project built as part of my freeCodeCamp coding practice.

## About the Project

CatPhotoApp is a beginner-friendly web page created using basic HTML. It includes:

- Headings
- Paragraphs
- Links
- Images
- Unordered and ordered lists
- Figure and figcaption elements
- Basic page structure using semantic HTML

This project helped me practise the foundations of web development and understand how to structure a simple webpage.

## Features

- A page about cat photos
- Links to cat-related websites and images
- A list of things cats love
- A list of things cats hate
- Images with descriptive alt text
- Use of semantic HTML elements like `main`, `section`, `figure`, and `footer`

## Built With

- HTML5

## What I Learned

While building this project, I practised:

- Creating a basic HTML document
- Using headings and paragraphs
- Adding links with the anchor tag
- Displaying images with the image tag
- Creating ordered and unordered lists
- Using semantic HTML for better page structure
- Writing alt text for accessibility
